CRM Privacy Policy

Effective Date:20-04-2026

This Privacy Policy describes how Sri Chaitanya CRM ("we", "our", or "us") collects, uses, and protects information when users access and use our Customer Relationship Management (CRM) application.

1. Scope of Service

This CRM is designed for internal organizational use. Access is restricted to authorized users within registered organizations.

2. Information We Collect

  • User Information: Name, email address, phone number, role within organization
  • Customer Data: Contact details, interaction notes, and relationship history entered by users
  • Call Data: Call logs, timestamps, duration, and related metadata obtained via Airtel API integration
  • Usage Data: Login activity, system interactions, and audit logs
  • Device & Technical Data: IP address, browser type, operating system
  • Profile Media: Profile photos uploaded by users

3. Permissions and Device Access

Our application may request access to your device storage or photo library solely for the purpose of allowing you to upload or update your profile picture.

  • Access is requested only when you choose to update your profile photo
  • We do not access, scan, or collect other media files from your device
  • Uploaded images are used only for profile display within the CRM

4. Authentication and Security

We use JSON Web Token (JWT) based authentication to securely manage user sessions. Access to data is restricted based on organizational roles and permissions.

5. How We Use Information

  • To provide and operate CRM functionalities
  • To manage customer relationships and communication records
  • To track and analyze call activity through Airtel API integration
  • To display user profile information including profile images
  • To improve system performance and security
  • To maintain audit logs and prevent unauthorized access

6. Data Sharing and Access Control

We do not sell personal data. Data access is limited as follows:

  • Accessible only within the same organization
  • Shared with service providers strictly for infrastructure and functionality
  • Disclosed if required by law or regulatory authorities

7. Third-Party Services

We integrate with third-party services such as Airtel API for call data retrieval. These services operate under their own privacy policies.

8. Data Storage and Backend

All data is securely stored and processed using our backend infrastructure built with Django.

9. Data Retention

We retain data only as long as necessary for operational, legal, and compliance purposes.

10. User Rights

  • Access and review their data
  • Request correction of inaccurate data
  • Request deletion where applicable

11. Cookies and Tracking

We may use cookies or similar technologies for authentication and improving user experience.

12. Children’s Privacy

This CRM is not intended for individuals under 18.

13. Changes to This Policy

We may update this Privacy Policy periodically.

Data Deletion

Users can request account and data deletion by contacting us at saransai.b@varsitymgmt.com. Upon request, we will delete all associated data within a reasonable timeframe.
